# Code of Conduct

## Purpose

This repository is created for learning and academic discussion.  
Everyone is expected to interact respectfully and help maintain a positive learning environment.

---

## Where This Applies

This Code of Conduct applies to:
- Issues
- Pull requests
- Discussions
- Comments
- Any interaction related to this repository

---

## Expected Behavior

Please:
- Be polite and respectful
- Discuss ideas and solutions, not people
- Accept corrections and feedback positively
- Share knowledge honestly
- Give credit when using someone elseâ€™s work

---

## Unacceptable Behavior

Do not:
- Use abusive, hateful, or offensive language
- Personally attack or insult others
- Spam, troll, or intentionally disrupt discussions
- Copy content without proper credit
- Share private information without permission

---

## Reporting Problems

If you notice behavior that breaks these rules:
- Open an issue (if appropriate), or
- Contact the repository owner privately

All reports will be handled fairly.

---

## Enforcement

If the rules are broken, the repository owner may:
- Give a warning
- Remove comments or contributions
- Restrict or block participation if necessary

---

## Final Note

This project is for learning.  
Be kind, be honest, and help others learn.
